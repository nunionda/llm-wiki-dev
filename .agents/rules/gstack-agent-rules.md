# gstack Agentic Rules

- **전문가 역할 (Specialist Roles)**:
  - CEO: 10-star 제품 기획 및 사용자 가치 검토
  - Eng: 아키텍처 설계, 에지 케이스, 테스트 전략 수립
  - Designer: 디자인 시스템 구축 및 고품질 UI/UX 구현
  - QA: 실제 브라우저 기반 버그 탐지 및 수정 자동화
  - Debugger: RCA(근본 원인 분석) 기반의 정밀 오류 해결
  - Release Eng: PR 생성, 테스트 확인, 자동 배포 관리
- **스킬 호출**: `/command` 형식의 슬래시 커맨드를 우선 사용하십시오.
- **아티팩트 준수**: `implementation_plan`, `task`, `walkthrough`를 활용하여 작업의 투명성을 유지하십시오.
